# Elements Learning App

## Deutsch

Interaktive Lernwebseite zum Periodensystem fuer Kinder und Jugendliche.

### Varianten

- Variante 1 ([index.html](index.html)): Automatisches Atommodell nach Auswahl eines Elements.
- Variante 2 ([lernmodus.html](lernmodus.html)): Lernmodus zum aktiven Ausfuellen und schrittweisen Aufbau eines Atoms.

### Was man lernen kann

- Ordnungszahl, Atommasse, Periode, Hauptgruppe
- Zusammenhang: Protonen = Ordnungszahl
- Neutronen-Berechnung: Neutronen = Masse - Protonen
- Elektronenverteilung auf Schalen
- Valenzelektronen in Hauptgruppen

### Lernhilfe im Lernmodus

Neben jedem Eingabefeld gibt es einen kleinen Nothilfe-Button.
Beim Klick werden passende Stellen im PSE kurz hervorgehoben (inkl. Pfeil) und eine kurze Erklaerung angezeigt.

### Start

Direkt im Browser oeffnen:

- [index.html](index.html)
- [lernmodus.html](lernmodus.html)

Es ist kein Build-Schritt und kein Server erforderlich.

### iPhone-Version (teilen per Link)

Die App wird automatisch als GitHub Pages Seite veroeffentlicht:

- https://kennerblick.github.io/elements/

Auf dem iPhone in Safari oeffnen und dann:

1. Teilen-Symbol antippen
2. Zum Home-Bildschirm
3. Namen festlegen, Hinzufuegen

So kann dein Kind die App wie eine normale App starten.

## English

Interactive periodic table learning web app for children and teenagers.

### Modes

- Mode 1 ([index.html](index.html)): Automatic atom model after selecting an element.
- Mode 2 ([lernmodus.html](lernmodus.html)): Hands-on learning mode to fill values and build the atom step by step.

### Learning goals

- Atomic number, atomic mass, period, main group
- Relation: protons = atomic number
- Neutron calculation: neutrons = mass - protons
- Electron shell distribution
- Valence electrons in main-group elements

### Help buzzer in learning mode

Each input field has a small help button.
On click, the app briefly highlights the matching location in the periodic table (with an arrow) and shows a short explanation.

### Start

Open directly in a browser:

- [index.html](index.html)
- [lernmodus.html](lernmodus.html)

No build step and no server required.

### iPhone version (share via link)

The app is automatically published as a GitHub Pages site:

- https://kennerblick.github.io/elements/

On iPhone, open it in Safari and then:

1. Tap Share
2. Add to Home Screen
3. Set a name and tap Add

This lets your child launch it like a regular app icon.
